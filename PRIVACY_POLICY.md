# VANIR Mobile Privacy Policy (Draft)

Last updated: 2026-06-26

## 1) Scope

This Privacy Policy describes how VANIR GROUP collects, uses, and protects data when you use the VANIR Mobile application.

## 2) Data We Collect

- Account identity data (name, email) from Google Sign-In and Firebase Authentication.
- Booking and service request data you submit (trip details, traveler info, visa request details, insurance details).
- Uploaded visa documents submitted through the in-app visa flow.
- Technical usage and diagnostics data required for app reliability and security.

## 3) Why We Use Data

- To authenticate users and secure sessions.
- To process bookings, visa assistance, eSIM activation, and insurance checkout.
- To provide customer support and travel operations.
- To improve performance, security, and fraud prevention.

## 4) Data Sharing

We only share data with service providers required to fulfill travel services (for example payment, visa, insurance, and connectivity partners), and only to the extent necessary for fulfillment and compliance.

## 5) Data Retention

We retain personal and transaction data only for the required business/legal duration and then delete or anonymize it.

## 6) Security

We apply industry-standard security controls for data transmission, access control, and storage.

## 7) User Rights

You can request access, correction, or deletion of your account data via:

- Email: info@vanirgroup.com
- App profile: Request Account Deletion action
